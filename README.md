# Projeto-e-implementacao-de-circuitos-sequenciais
Este projeto consiste na implementação de circuito sequencias para a criação de um dado digital e um semáforo.
O dado digital foi implementado no thinkerCad e o semáforo no digitalworks.
Está tudo devidamente relatado no relatório e em anexo ainda tem o dado digital realizado no thinkerCad.
